# Email Brief Template

For single sends, sequences, and automated email campaigns.

```markdown
---
title: [Email Campaign Name]
type: email
date: YYYY-MM-DD
send-type: [single|sequence|automation]
---

# [Email Campaign Name]

## Goal
[What this email should accomplish — open, click, reply, purchase, etc.]

## Audience
**Segment:** [Specific list segment or trigger condition]
**List size:** [Approximate count]
**Relationship:** [New subscriber / Active user / Churned / Trial / Paid]

## Subject Line Options (provide 3)
1. [Option 1 — under 50 characters]
2. [Option 2 — under 50 characters]
3. [Option 3 — under 50 characters]

**Subject line strategy:** [Curiosity gap / Benefit-led / Urgency / Question / Social proof]

## Preheader
[Text that appears after subject in inbox — complements but doesn't repeat subject, ~90 characters]

## Opening Line
[The hook — should pull directly from the subject line's promise]

## Core Message
[Single most important thing — the email should communicate this above all else]

## Body Structure
[How the email flows — sections or bullets]

1. **Hook:** [First 1-2 sentences that keep them reading]
2. **Problem/context:** [Brief setup]
3. **Core content:** [The value delivery]
4. **Bridge to CTA:** [Transition to the ask]

## Primary CTA
**Action:** [What we want them to click]
**CTA copy:** [Exact button or link text]
**URL:** [Destination URL]

## Secondary CTA (if any)
[Second option if primary isn't right for them]

## Tone
[3 adjectives or reference example]

## Send Details
- **From name:** [Name that appears in inbox]
- **Reply-to:** [Reply email address]
- **Send time:** [Day and time — with timezone]
- **Send date:** [Date]

## Sequence (if multi-email)
| Email # | Goal | Days after trigger | Subject |
|---------|------|-------------------|---------|
| 1 | | Day 0 | |
| 2 | | Day 3 | |
| 3 | | Day 7 | |

## Success Metrics
- **Open rate goal:** [%]
- **CTR goal:** [%]
- **Primary conversion goal:** [metric + target]
```
